lifx-rest
=========

Minimal rest server using nodejs/restify.


**Installation**

```bash
git clone https://github.com/MrRacoon/lifx-rest.git
cd lifx-rest
npm install
```

**Running**

`npm run start`



**EndPoints**


| Method  | route  | action |
|:--------|:-------|:------:|
| GET     | /bulbs | Get the current state of bulbs that are currently seen by the server |
